🚑 Real-Time Ambulance Detection using OCR

📌 Overview



This project detects ambulances in real-time using Optical Character Recognition (OCR). It identifies the text “AMBULANCE” (including reversed text) and simulates traffic signal prioritization.



🎯 Features

Real-time detection using webcam

Text recognition using EasyOCR

Detects reversed text ("ECNALUBMA")

Confidence-based filtering

On-screen detection display

Simulated traffic signal logic

🛠️ Tech Stack

Python

OpenCV

EasyOCR



▶️ How to Run

pip install -r requirements.txt

python detector.py
